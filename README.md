# articulatelearning.org

Marketing and legal website for the **Articulate** iOS app by Real Intelligence Tech LLC.

## Pages

| Route | Description |
|---|---|
| `/` | Landing page |
| `/terms` | Terms of Use |
| `/privacy` | Privacy Policy |

## Stack

- React 19 + TypeScript
- React Router v7
- Vite + Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Assets

Place app assets in `public/`:
- `articulate-icon.png` — app icon (also used as favicon)
- `screenshots/01.png` – `06.png` — carousel screenshots
